## 📌 프로젝트 구조

### Client

- `components`: 프로젝트에서 사용되는 컴포넌트
- `pages`: 프로젝트에서 사용되는 페이지
- `services`: 데이터를 받아오는 api 관련 로직
- `style`: 스타일 코드 파일
- `App.jsx`: 해당 프로젝트 루트 파일
